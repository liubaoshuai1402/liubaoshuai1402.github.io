и¶…жҷ¶ж јжқҗж–ҷеҺҹеӯҗе°әеәҰе»әжЁЎ

еүҚиЁҖ

иҷҪз„¶ејӮиҙЁз»“зҡ„е»әжЁЎж”»з•ҘеҫҲеӨҡпјҢдҪҶеӨ§йғҪд»ҘMaterial Studioдёәдё»пјҢдё”еҜ№иұЎжҳҜиЎЁз•ҢйқўгҖӮеҜ№дәҺе‘ЁжңҹжҖ§зҡ„и¶…жҷ¶ж јжқҗж–ҷзҡ„е»әжЁЎпјҢзү№еҲ«жҳҜејӮиҙЁз»“жғ…еҶөпјҢеҸӮиҖғиҫғе°‘пјҢиҝҷйҮҢжҲ‘еҲҶдә«дёҖдёӢдёӘдәәзҡ„з»ҸйӘҢгҖӮйҮҮз”ЁASEиҝӣиЎҢе»әжЁЎгҖӮ

ејӮиҙЁз»“зҡ„дёӨз§Қжқҗж–ҷжҺҘи§Ұж—¶пјҢе“ӘдёӨдёӘйқўй—ҙзҡ„жҺҘи§ҰпјҢжҳҜйңҖиҰҒзЎ®е®ҡзҡ„пјҢжҜ”еҰӮиҜҙпјҢж №жҚ®е®һйӘҢзЎ®е®ҡпјҢеҸҲжҲ–иҖ…ж— е®һйӘҢж—¶пјҢж №жҚ®жҷ¶ж јеҢ№й…ҚеәҰзЎ®е®ҡпјҢе°ҪйҮҸдҝқиҜҒеӨұй…ҚеәҰиҫғдҪҺгҖӮ

иҝҷйҮҢпјҢеҒҮи®ҫе·Із»ҸзЎ®е®ҡдәҶпјҢдёӨдёӘжқҗж–ҷзҡ„жҷ¶иғһиҰҒжІҝзқҖzиҪҙе ҶеҸ гҖӮ

ж•ҙдҪ“жҖқи·Ҝпјҡ

1.зЎ®е®ҡеҘҪиҰҒеҗҲе№¶зҡ„дёӨдёӘжҷ¶ж јзҡ„е…·дҪ“з»“жһ„пјҲз”Ёtranslateе№із§»пјҢsurfaceеҲҮйқўпјҢиҝҷдёӘйЎәеәҸеҘҪеғҸд№ҹиғҪеҸҚиҝҮжқҘпјү

2.еҗҲе№¶жҷ¶ж јпјҲstackз”Ёиө·жқҘиҝҳжҳҜиӣ®йңҖиҰҒз»ҸйӘҢзҡ„пјү

еҰӮдҪ•еҲҮдёҖдёӘжҷ¶йқўпјҢе№¶з”ҹжҲҗе‘ЁжңҹжҖ§з»“жһ„

from ase.build import surface,stack,make_supercell
from ase.io import read,write
from ase.io.vasp import write_vasp

BiTe = read('POSCAR_BiTe')
BiTe = make_supercell(BiTe,P=[[2,0,0],[0,1,0],[0,0,1]])
BiTe1 = BiTe.copy()
BiTe2 = BiTe.copy()
BiTe2.translate([0,0,-2.05985])
BiTe3 = BiTe.copy()
BiTe3.translate([0,0,-3.79553])

BiTe1 = surface(BiTe1,indices=(0,0,1),layers=1,periodic=True)
BiTe2 = surface(BiTe2,indices=(0,0,1),layers=1,periodic=True)
BiTe3 = surface(BiTe3,indices=(0,0,1),layers=1,periodic=True)
write_vasp('POSCAR_BiTe1',BiTe1,direct=True,sort=True)
write_vasp('POSCAR_BiTe2',BiTe2,direct=True,sort=True)
write_vasp('POSCAR_BiTe3',BiTe3,direct=True,sort=True)

aseзҡ„surfaceеҮҪж•°еҸҜд»ҘеҫҲз®ҖеҚ•зҡ„е®һзҺ°пјҢиҝҷйҮҢдёҚиөҳиҝ°дәҶгҖӮ

жӯӨеӨ–пјҢе…ідәҺеҰӮдҪ•зЎ®е®ҡдёҖдёӘеҗҲйҖӮзҡ„ж–°жҷ¶ж јзҡ„еӨ§е°ҸпјҢд№ҹжңүеҫҲеӨҡи§Ҷйў‘и®Іи§ЈпјҢдёҚеҶҚиөҳиҝ°гҖӮ

ж”№еҸҳжҷ¶дҪ“з»“жһ„зҡ„еҺҹеӯҗеұӮйЎәеәҸ

еҚідҫҝжҳҜеҸӘжңүдёҖз§ҚеҺҹеӯҗзҡ„жҷ¶дҪ“пјҢе…¶жІҝжҹҗдёӘйқўзҡ„е Ҷеһӣзҡ„ж—¶еҖҷпјҢеҸҜиғҪжңүдёҚеҗҢзҡ„еұӮгҖӮжҜ”еҰӮFCCжҷ¶дҪ“жІҝ001йқўзҡ„е Ҷеһӣж–№ејҸе°ұжҳҜ—ABAB—пјҢжҳҫз„¶дёҺеҲ«зҡ„зү©иҙЁеҪўжҲҗејӮиҙЁз»“ж—¶пјҢдјҡйқўдёҙдёҖдёӘй—®йўҳпјҢеҚіжҳҜAйқўиҝҳжҳҜBйқўдёҺеҲ«зҡ„зү©иҙЁжҺҘи§ҰгҖӮ

жҲ‘д»¬е»әжЁЎж—¶пјҢйңҖиҰҒжҠҠAйқўжҲ–Bйқўи°ғж•ҙеҮәжқҘгҖӮиҝҷйңҖиҰҒеҜ№еҺҹеӯҗиҝӣиЎҢж•ҙдҪ“дҪҚз§»

жӯӨеӨ–пјҢејӮиҙЁз»“е№ійқўеҶ…зҡ„еҺҹеӯҗеҜ№йҪҗпјҲжҜ”еҰӮиҜҙxyйқўпјүпјҢд№ҹйңҖиҰҒеҜ№еҺҹеӯҗиҝӣиЎҢж•ҙдҪ“дҪҚз§»гҖӮ

иҖҢASEе®һзҺ°еҺҹеӯҗж•ҙдҪ“дҪҚз§»йқһеёёз®ҖеҚ•пјҢеҸӘйңҖиҰҒз”Ёatomsзұ»зҡ„translateж–№жі•еҚіеҸҜпјҢжіЁж„ҸдҪҝз”Ёз»қеҜ№еқҗж ҮгҖӮ

from ase.io import read,write
from ase.io.vasp import write_vasp

BiTe = read('POSCAR_BiTe')
BiTe = make_supercell(BiTe,P=[[2,0,0],[0,1,0],[0,0,1]])
BiTe1 = BiTe.copy()
BiTe2 = BiTe.copy()
BiTe2.translate([0,0,-2.05985])

ASEе®һзҺ°и¶…жҷ¶ж јжқҗж–ҷе»әжЁЎ

from ase.build import surface,stack,make_supercell
from ase.io import read,write
from ase.io.vasp import write_vasp

BiTe1 = read('POSCAR_BiTe1')
BiTe2 = read('POSCAR_BiTe2')
BiTe3 = read('POSCAR_BiTe3')

Ag = make_supercell(read('POSCAR_Ag'),P=[[2,0,0],[0,2,0],[0,0,3]])
BiTe1_Ag = stack(Ag,BiTe1,axis=2,fix=0.5)
BiTe2_Ag = stack(Ag,BiTe2,axis=2,fix=0.5)
BiTe3_Ag = stack(Ag,BiTe3,axis=2,fix=0.5)

write('POSCAR_BiTe1_Ag',BiTe1_Ag,format='vasp')
write('POSCAR_BiTe2_Ag',BiTe2_Ag,format='vasp')
write('POSCAR_BiTe3_Ag',BiTe3_Ag,format='vasp')
е…ідәҺaseдёӯstackеҮҪж•°зҡ„еҺҹзҗҶгҖӮ

е°ҶдёӨдёӘе‘ЁжңҹжҖ§зҡ„жҷ¶иғһеҗҲе№¶жҲҗдёҖдёӘе‘ЁжңҹжҖ§зҡ„жҷ¶иғһпјҢжңүдёҖдёӘеҝ…йЎ»иҰҒйқўеҜ№зҡ„й—®йўҳпјҢйӮЈе°ұжҳҜжҺҘи§Ұйқўе°ҶдёҚеҶҚжңүе‘ЁжңҹжҖ§гҖӮ

з®ҖеҚ•жқҘи®ІпјҢеҺҹжң¬зҡ„еӣӣдёӘе‘ЁжңҹжҖ§йқўпјҢзҺ°еңЁд»…еү©дҪҷдёӨдёӘгҖӮ

з„¶еҗҺпјҢдҪңдёәжҷ¶дҪ“з»“жһ„ж–Үд»¶зҡ„жғҜдҫӢпјҢжҲ‘д»¬жҖ»жҳҜжҠҠеҺҹеӯҗж”ҫеңЁе‘ЁжңҹжҖ§йқўдёҠпјҢеҚіжҷ¶ж јиҫ№зјҳеӨ„пјҢиҝҷж ·еңЁжҳҫзӨәж—¶пјҢдјҡжңүйҮҚеӨҚжҳҫзӨәзҡ„жғ…еҶөгҖӮ

е…·дҪ“жқҘи®ІпјҢеҰӮеӣҫпјҢзәўиүІж јеӯҗе’Ңи“қиүІж јеӯҗпјҢе…¶е®һеҸӘжңүдёӨеұӮеҺҹеӯҗпјҢдёҖеұӮzеқҗж Үдёә0.0пјҲеҲҶеқҗж ҮпјүпјҢдёҖеұӮzдёә0.5пјҢдҪҶжҳҫзӨәж—¶пјҢzдёә1зҡ„жғ…еҶөд№ҹеӣ дёәе‘ЁжңҹжҖ§жҳҫзӨәеҮәжқҘдәҶгҖӮ

stackеҗҲе№¶ж—¶пјҢе…¶е®һжҳҜжҠҠи“қиүІж јеӯҗпјҢпјҲд№ҹе°ұжҳҜз¬¬дёҖдёӘatomsеҜ№иұЎпјүпјҢе®ғзҡ„z=1зҡ„е‘ЁжңҹжҖ§йқўпјҢдҝқз•ҷеҲ°дәҶжңҖдёҠж–№пјҢеҚіgoodжғ…еҶөгҖӮ

йӮЈд№Ҳд»Җд№ҲжҳҜеқҸжғ…еҶөе‘ўпјҢеқҸжғ…еҶөжҳҜпјҢзәўиүІж јеӯҗзҡ„жҷ¶дҪ“з»“жһ„ж–Үд»¶дёӯпјҢдёӨеұӮеҺҹеӯҗеқҗж Үдёәz=0.0пјҢz=1.0гҖӮ

иҝҷз§Қbadжғ…еҶөе°ұдјҡеҜјиҮҙжҸҗеҲ°дёҠж–№зҡ„зәўиүІеҺҹеӯҗдёҺи“қиүІеҺҹеӯҗеҮәзҺ°еңЁеҗҢдёҖе№ійқўдёҠпјҢдёҚжҳҜжҲ‘д»¬жңҹжңӣзҡ„жғ…еҶөгҖӮ

Important

жүҖд»ҘпјҢиҜ·дҝқиҜҒз¬¬дәҢдёӘatomsдёӯпјҢz=1.0жҲ–иҖ…еҮ д№ҺжҺҘиҝ‘дәҺ1.0зҡ„еҺҹеӯҗпјҢе…ЁйғЁйҖҡиҝҮе‘ЁжңҹжҖ§жҚўз®—жҲҗ0пјҢиҝҷеҫҲйҮҚиҰҒпјҒ

е…¶е®һе°ұжҳҜпјҢдёӨдёӘжҷ¶иғһзҡ„еҺҹеӯҗз»“жһ„ж–Үд»¶дёӯпјҢеҺҹеӯҗе Ҷз Ңж–№еҗ‘еә”иҜҘдёҖиҮҙпјҢдёҚиғҪдёҖдёӘд»ҺдёӢе Ҷз ҢеҲ°дёҠпјҢиҖҢеҸҰдёҖдёӘд»ҺдёҠе Ҷз ҢеҲ°дёӢгҖӮ

е…ідәҺdistanceеңЁжҺ§еҲ¶дәӣд»Җд№Ҳ

distanceе…¶е®һжҳҜжҺ§еҲ¶дёӨдёӘжҺҘи§Ұйқўй—ҙејӮиҙЁз»“еҺҹеӯҗзҡ„жңҖе°Ҹи·қзҰ»пјҢдјҡеҪұе“ҚжңҖз»Ҳcellзҡ„zж–№еҗ‘е°әеҜёд»ҘеҸҠеҜ№еҺҹеӯҗиҝӣиЎҢеҫ®жү°пјҢжүҖд»ҘжңҖдёҠе’ҢжңҖдёӢзҡ„дёӨдёӘйқўдёҠзҡ„еҜ№з§°еҺҹеӯҗеҸҜиғҪз”ұдәҺе‘ЁжңҹжҖ§пјҢеҸҳжҲҗдёҖеұӮгҖӮ
