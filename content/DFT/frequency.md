# 零点能计算

## 前言

我对零点能的理解非常肤浅，以下仅是从网上捕风捉影而来，用于个人记录。

在计算自由能时，看到一些文献的做法是：DFT总能+零点能+熵。

零点能是0K下，体系的振动能，可以通过声子频率计算而来。

声子计算可能由phonopy完成，但是也有通过单vasp计算的。

这里记录一下，只用vasp计算gamma点声子频率的步骤，用于零点能的粗略计算。

### INCAR

```
ISMEAR = 0
IBRION = 5  
NFREE = 2  
POTIM = 0.02
NSW = 1
```

频率计算的关键参数。

`IBRION = 5`启用有限位移法计算声子模。

`NFREE = 2`  设置原子振动的自由度。

`POTIM = 0.02` 设置振动的幅度。

`NSW = 1` 离子步为1
